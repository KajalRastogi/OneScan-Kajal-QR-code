## Project Overview
**OneScan Kajal QR Code** is a lightweight and efficient tool that allows users to **generate custom QR codes** for websites, videos, or files in seconds. Developed in **Python** using `qrcode` and `Pillow`, the project focuses on creating **reliable, scannable QR codes** with customizable design options. This tool is perfect for **instant digital sharing**, hackathon demos, or prototype applications.  

---

## Features
- Generate QR codes instantly with a single input  
- Save QR codes as images in PNG format  
- Customize **size, border, and error correction level**  
- Easily extendable for **dynamic QR codes, analytics, and web interfaces**  

---

## Technology Stack
- **Programming Language:** Python  
- **Libraries:** `qrcode`, `Pillow`  
- **Development Platform:** VS Code / Windows  
- **Potential Enhancements:** Streamlit or Flask (web interface), AWS S3 / Google Drive (cloud storage), SQLite or Firebase (database)
